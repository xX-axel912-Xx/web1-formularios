10- Qué son los métodos / verbos de petición HTTP? Dar ejemplos. Bibliografía MDN

RTA: HTTP define un conjunto de métodos de petición para indicar la acción que se desea realizar para un recurso determinado. Aunque estos también pueden ser sustantivos, estos métodos de solicitud a veces son llamados HTTP verbs, algunos ejemplos de estos son: get, head, post, put, ETC

11- Qué son los códigos de estado de respuestas HTTP? Realizar una tabla con su clasificación

Los códigos de estado de respuesta HTTP indican si se ha completado satisfactoriamente una solicitud HTTP

codigos de estado de respuesta HTTP
respuestas informativas(100-199) respuestas satisfactorias(200-299) redirecciones(300-399) errores del cliente(400-499) errores del servidor(500-599)
